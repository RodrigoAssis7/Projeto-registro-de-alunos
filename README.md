# 🎓 Sistema de Registro de Alunos

Sistema de gerenciamento de alunos desenvolvido em Python, com foco em organização de código, persistência de dados e aplicação prática de Programação Orientada a Objetos (POO).

O projeto simula um pequeno sistema acadêmico em ambiente CLI (Command Line Interface), permitindo cadastro, busca, listagem e remoção de alunos com armazenamento local de dados.

---

## Funcionalidades

- Cadastro de alunos
- Busca por nome ou matrícula
- Listagem completa de registros
- Remoção de alunos com confirmação
- Persistência automática em arquivo `.csv`
- Validação de entradas
- Tratamento de erros e exceções
- Estrutura modularizada

---

## Tecnologias Utilizadas

- Python 3
- Programação Orientada a Objetos (POO)
- Manipulação de arquivos CSV
- Estruturas condicionais e de repetição
- Modularização em múltiplos arquivos
- Tratamento de exceções (`try/except`)

## Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/RodrigoAssis7/Projeto-registro-de-alunos.git
```

### 2. Acesse a pasta do projeto

```bash
cd Projeto-registro-de-alunos
```

### 3. Execute o sistema

```bash
python3 main.py
```

---

## Armazenamento de Dados

Os dados dos alunos são armazenados localmente em arquivos `.csv`, permitindo persistência mesmo após o encerramento do sistema.
